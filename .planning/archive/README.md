# Archived plan documents

Implementation plans for shipped milestones, kept as historical
record. Each file is named `<version>-<slug>.md` and reflects the
plan as written before the milestone shipped — not a post-hoc
summary.

Files in this directory frequently reference `ROADMAP_STATUS.md`,
which was renamed to [`docs/roadmap.md`](../../docs/roadmap.md) when
these plans were archived. The contents are otherwise preserved
verbatim.

For a chronological tour of what each milestone actually shipped,
read [`docs/roadmap.md`](../../docs/roadmap.md) — the archives here
are the *plans*, the roadmap is the *outcome*.
